◆ R_BSP_CacheInvalidate()

void R_BSP_CacheInvalidate ( void  )

Invalidates instruction and data caches of arm core processor. Use the MRC or MCR instructions to access coprocessor registers. If you access the coprocessor with a C compiler, use inline assembly language. The cache invalidation process is defined in bsp_cache.c for each device.
