Detailed Description

Driver for phase count using the MTU3 peripheral on RZ MCUs. This module implements the Phase count Interface.

Overview

Multi-Function Timer Pulse Unit 3 (MTU3) Phase Count Driver provides the basic functionality of the PWM channel used in phase counting applications. The ability to set the count value of the channel and the function to obtain the count value are possible.

Features

The MTU3 phase count driver provides the following functions:

  • 16-bit phase counting mode supports 5 types of phase counting modes
  • The start, stop and reset MTU3 channels
  • The get, set of MTU3 phaseCount
  • In 16-bit mode, it has a function to detect the position and speed in cooperation with MTU0
  • In 32-bit mode, it has a function to capture Z-phase signal

Usage Notes

MTU3 phase count Basic Example

This is a basic example of minimal use of the MTU3 phase count module in an application.

void mtu3_callback (timer_callback_args_t * p_args)
{
fsp_err_t err;
if (p_args != NULL)
{
/* Get location information (Phase Count) */
err = R_MTU3_PHASE_COUNT_PositionGet(&g_mtu3_phase_count_ctrl, g_mtu3_phase_count_get_pos);
handle_error(err);
}
}
void mtu3_phase_count_example (void)
{
fsp_err_t err = FSP_SUCCESS;
/* Initializes the module. */
err = R_MTU3_PHASE_COUNT_Open(&g_mtu3_phase_count_ctrl, &g_mtu3_phase_count_cfg);
/* Handle any errors. This function should be defined by the user. */
handle_error(err);
/* Start the timer. */
(void) R_MTU3_PHASE_COUNT_Start(&g_mtu3_phase_count_ctrl);
}
