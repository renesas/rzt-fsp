SDRAM Interface
Interfaces

Detailed Description

Interface for external sdram communication.

Summary

The SDRAM interface provides APIs and definitions for SDRAM communication.

Implemented by:

  • Bus State Controller (BSC) SDRAM (r_bsc_sdram)
