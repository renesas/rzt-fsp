Detailed Description

Driver for phase count using the GPT peripheral on RZ MCUs. This module implements the Phase count Interface.

Overview

The General PWM Timer (GPT) Phase Count Driver provides the basic functionality of the PWM channel used in phase counting applications. The ability to set the count value of the channel and the function to obtain the count value are possible.

Features

The GPT phase count driver provides the following functions:

  • The start, stop and reset GPT channels
  • The get, set of GPT phaseCount

Usage Notes

GPT phase count Basic Example

This is a basic example of minimal use of the GPT phase count module in an application.

void gpt_callback (timer_callback_args_t * p_args)
{
fsp_err_t err;
if (p_args != NULL)
{
/* Get location information (Phase Count) */
err = R_GPT_PHASE_COUNT_PositionGet(&g_gpt_phase_count_ctrl, g_gpt_phase_count_get_pos);
handle_error(err);
}
}
void gpt_phase_count_example (void)
{
fsp_err_t err = FSP_SUCCESS;
/* Initializes the module. */
err = R_GPT_PHASE_COUNT_Open(&g_gpt_phase_count_ctrl, &g_gpt_phase_count_cfg);
/* Handle any errors. This function should be defined by the user. */
handle_error(err);
/* Start the timer. */
(void) R_GPT_PHASE_COUNT_Start(&g_gpt_phase_count_ctrl);
}
