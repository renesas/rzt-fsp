POE3 Interface
Interfaces

Detailed Description

Interface for the Port Output Enable for MTU3.

Defines the API and data structures for the Port Output Enable for MTU3 (POE3) interface.

Summary

The POE3 disables MTU3 output pins based on configurable events.

Implemented by: Port Output Enable 3 (r_poe3)
