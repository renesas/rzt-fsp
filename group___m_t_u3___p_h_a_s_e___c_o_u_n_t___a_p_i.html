Phase count Interface
Interfaces

Detailed Description

Interface for phase count timer functions.

Summary

The Phase count interface provides functionality for synchronous start/stop/reset control of three timer channels for use in phase count counter applications.

Implemented by:

  • Multi-Function Timer Pulse Unit 3 phase count Motor Control Driver (r_mtu3_phase_count)
